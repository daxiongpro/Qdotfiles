### 选择节点的网站：

http://clash.razord.top/#/proxies

### useage:

* make sure clash config exist.
  `qdot b`
* download config file

  ```bash
  cd ~/clash
  wget xxx.yaml # your clash config file. go to clash website to buy it.
  ```
* use tmux:

  ```
  tmux new -s proxy
  proxy up
  ```
* port is 8999
